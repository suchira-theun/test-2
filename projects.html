IEEE-Published Research

STEALTH EYE

Behavioral Analysis for Fileless Malware Detection

A real-time endpoint behavioral-analysis framework designed to detect ransomware, RedLine Stealer, spyware, and trojans that operate without relying on conventional file-based signatures.

The system continuously captures memory behavior, DLL activity, process and thread events, registry modifications, network communication, cryptographic operations, keystroke activity, and clipboard access. Supervised machine-learning models analyze these behavioral snapshots to identify malicious patterns and generate actionable alerts.

01

Research Problem

Detecting threats that leave no traditional file signature

Fileless malware executes through memory and trusted system tools, enabling it to evade conventional signature-based and file-centric security controls.

02

Proposed Solution

Continuous behavioral monitoring with machine learning

Endpoint agents capture structured behavioral snapshots every 60 seconds and evaluate them using supervised models trained to distinguish benign and malicious activity.

03

Research Outcome

Real-time alerting, forensic logging, and active response

The framework delivers threat probabilities, preserves behavioral evidence, securely transfers alerts, and visualizes events through a centralized administrative dashboard.

System Workflow

From endpoint activity to security response

Four connected stages transform low-level system behavior into actionable threat intelligence.

01

Monitor

Observe memory, processes, DLLs, registry events, user activity, and network behavior.

02

Capture

Generate structured endpoint snapshots at defined monitoring intervals.

03

Analyze

Preprocess behavioral features and classify threats using trained machine-learning models.

04

Alert

Log evidence, transmit threat data securely, and support rapid incident response.
